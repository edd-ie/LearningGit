To initiate git:
                 >Right click on the folder and press Git Bash
                 >Type [ git init ] = create a git repository ==> {done once}
                 ------------------------------------------------------------------
                 >Type [ git add . ] / [ git stage ] = the dot adds all files
                 >Type [ git status ] = show the current status of file---={ not important }
                 >Type [ git commit -m "relevant comment" ] = records changes of repository
                 >Go to github and click on [ + sign] then [ add new repository]
                 >Add the name of the Repository
                 >Select if public/private
                 >Click on [ Create repository ]
                 >Paste the Line of code under push file into git and enter
                 >Refresh webpage
